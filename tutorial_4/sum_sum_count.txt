Count the number of digits in an input number.

num = 12
n = 1
sum = 0
while(n<num+1):
	sum = sum + n
	n = n + 1

print(sum)



Sum all the digits in an input number.

num = 438
sum = 0
while(num>0):
	last_digit = num % 10
	num = num // 10
	sum = sum + last_digit
